# Lab 16 - Employee Management Application

![CI](https://github.com/kedarnhegde/CS648-lab/workflows/CI/badge.svg?branch=lab-16)

Employee Management Application with React, Bootstrap, and MongoDB.

## Features
- Add/Delete employees
- Filter by employment status
- Bootstrap UI with modals
- MongoDB integration

## Setup
```bash
npm install
```

## Development
```bash
# Terminal 1 - Watch and build
npm run watch

# Terminal 2 - Start server
npm start
```

## Testing
```bash
# Run tests
npm test

# Run linter
npm run lint
```

## CI/CD
This project uses GitHub Actions for:
- ESLint code quality checks
- Jest unit tests
- Automated testing on push/PR
